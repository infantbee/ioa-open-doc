## 1. 接口描述

查询数据地图策略适用范围，私有化调用path为：/capi/DLPOpenApi/DescribeBackendPolicyScope，从8.6P1版本开始支持

## 2. 输入参数

| 参数名称 | 必选 | 类型 | 描述 |
|---------|---------|---------|---------|
| PolicyId | 是 | Integer | 策略ID(只支持32位)<br/>示例值：1 |
| OsType | 是 | Integer | 系统类型(只支持32位)   0 Win  2 macOS<br/>示例值：0 |
| DomainInstanceId | 否 | String | 管理域实例ID，用于CAM管理域权限分配。若企业未进行管理域的划分，可直接传入根域"1"，此时表示针对当前企业的全部设备和账号进行接口CRUD，具体CRUD的影响范围限制于相应接口的入参。<br/>示例值：1 |
| AssociatedObject | 否 | String | 管控对象名称<br/>示例值：忽略 |

## 3. 输出参数

| 参数名称 | 类型 | 描述 |
|---------|---------|---------|
| Data | Array of [BackendPolicyScopeData](/开放API/云规范接口/版本：2022-06-01/数据结构.md#BackendPolicyScopeData) | 策略适用范围数据<br/>注意：此字段可能返回 null，表示取不到有效值。|
| RequestId | String | 唯一请求 ID，由服务端生成，每次请求都会返回（若请求因其他原因未能抵达服务端，则该次请求不会获得 RequestId）。定位问题时需要提供该次请求的 RequestId。|
