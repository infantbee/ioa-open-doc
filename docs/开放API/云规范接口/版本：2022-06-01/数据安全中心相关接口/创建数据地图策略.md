## 1. 接口描述

创建数据地图策略，私有化调用path为：capi/DLPOpenApi/CreateBackendPolicy，从8.6P1版本开始支持

## 2. 输入参数

| 参数名称 | 必选  | 类型 | 描述                                  |
|---------|-----|---------|-------------------------------------|
| Name | 是   | String | 策略名称;Required:true<br/>示例值：扫描       |
| Description | 否   | String | 描述<br/>示例值：扫描办公设备                   |
| Status | 否   | Integer | 启用停用状态 0停用 1启用(只支持32位)<br/>示例值：1    |
| Data | 是   | String | 策略数据,传空json对象,示例值："{}"              |
| BusinessId | 否   | Integer | 业务Id;(只支持32位)  320 数据地图<br/>示例值：320 |

## 3. 输出参数

| 参数名称 | 类型 | 描述 |
|---------|---------|---------|
| Data | [CreateBackendPolicyData](/开放API/云规范接口/版本：2022-06-01/数据结构.md#CreateBackendPolicyData) | 业务响应数据<br/>注意：此字段可能返回 null，表示取不到有效值。|
| RequestId | String | 唯一请求 ID，由服务端生成，每次请求都会返回（若请求因其他原因未能抵达服务端，则该次请求不会获得 RequestId）。定位问题时需要提供该次请求的 RequestId。|

## 4. 示例

### 示例1 创建数据地图扫描

#### 输入示例

```
POST / HTTP/1.1
Host: ioa.tencentcloudapi.com
Content-Type: application/json
X-TC-Action: CreateBackendPolicy
<公共请求参数>

{
  "Name": "test",
  "Status": 1,
  "Description": "",
  "Data": "{}"
}
```

#### 输出示例

```json
{
    "Response": {
        "Data": {
            "PolicyId": 49
        },
        "RequestId": "web-1245921911577-1765938455417"
    }
}
```
