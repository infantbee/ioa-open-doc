## 1. 接口描述

修改数据地图策略状态，私有化调用path为：/capi/DLPOpenApi/ModifyBackendPolicyStatus，从8.6P1版本开始支持

## 2. 输入参数
| 参数名称 | 必选 | 类型 | 描述 |
|---------|---------|---------|---------|
| Status | 是 | Integer | 通道状态 0:停用 1启用(只支持32位)<br/>示例值：1 |
| PolicyId.N | 是 | Array of Integer | 策略ID列表(只支持32位)<br/>示例值：[1] |

## 3. 输出参数

| 参数名称 | 类型 | 描述 |
|---------|---------|---------|
| RequestId | String | 唯一请求 ID，由服务端生成，每次请求都会返回（若请求因其他原因未能抵达服务端，则该次请求不会获得 RequestId）。定位问题时需要提供该次请求的 RequestId。|

## 4. 示例

### 示例1 修改数据地图策略状态

#### 输入示例

```
POST / HTTP/1.1
Host: ioa.tencentcloudapi.com
Content-Type: application/json
X-TC-Action: ModifyBackendPolicyStatus
<公共请求参数>

{
  "Status": 0,
  "PolicyId": [
    49
  ]
}
```

#### 输出示例

```json
{
    "Response": {
        "RequestId": "web-49563164851-1765953259317"
    }
}
```
