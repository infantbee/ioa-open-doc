## 1. 接口描述

查询满足条件的终端数据详情，私有化调用path为：/capi/Assets/Device/DescribeDevices，适用于私有化8.0以后的版本；8.0以前版本的调用path为：/capi/Assets/DescribeDevices。

## 4. 示例

### 示例1 获取设备列表详情

```
POST / HTTP/1.1
Host: ioa.tencentcloudapi.com
Content-Type: application/json
X-TC-Action: DescribeDevices
<公共请求参数>

{
  "Condition": {
    "FilterGroups": [
      {
        "Filters": [
          {
            "Field": "IOAUserName",
            "Operator": "ilike",
            "Values": [
              "cc"
            ]
          }
        ]
      }
    ],
    "PageSize": 10,
    "PageNum": 1

}
```
