## 1. 接口描述

针对一条规则查询规则信息，私有化调用path为：/capi/DlpOpenApi/DescribeDataRule，从8.6P1版本开始支持

## 2. 输入参数

| 参数名称 | 必选 | 类型 | 描述 |
|---------|---------|---------|---------|
| Condition | 否 | [Condition](/开放API/云规范接口/版本：2022-06-01/数据结构.md#Condition) | <strong><font color="blue"></font></strong>过滤条件<br> <li>Id - int - 是否必填：否 - 操作符: eq  - 排序支持：否- 按照规则Id进行过滤。</li> <li>RuleIds - []int - 是否必填：否 - 操作符: any  - 排序支持：否- 按照规则Id列表进行过滤。</li> <li>CategoryId - int - 是否必填：否 - 操作符: eq  - 排序支持：是- 按照分类Id进行过滤。</li> <li>Status - int - 是否必填：否 - 操作符: eq  - 排序支持：是- 按照启用状态进行过滤。</li> <li>IsCustomize - int - 是否必填：否 - 操作符: eq  - 排序支持：是- 按照自定义类型（0:内置 1自定义）进行过滤。</li> <li>Name - String - 是否必填：否 - 操作符: eq  - 排序支持：否- 按照规则名进行过滤。</li> <li>DataLevel - String - 是否必填：否 - 操作符: eq  - 排序支持：否- 按照数据等级进行过滤。</li> 分页参数<br> <li>PageNum 从1开始，小于等于0时使用默认参数1。</li> <li>PageSize 最大值5000，最好不超过100。</li> |

## 3. 输出参数

| 参数名称 | 类型 | 描述 |
|---------|---------|---------|
| Data | [DescribeDataRulePageData](/开放API/云规范接口/版本：2022-06-01/数据结构.md#DescribeDataRulePageData) | <strong><font color="blue">此参数仅产品内部展示。</font></strong>业务响应数据<br/>注意：此字段可能返回 null，表示取不到有效值。|
| RequestId | String | 唯一请求 ID，由服务端生成，每次请求都会返回（若请求因其他原因未能抵达服务端，则该次请求不会获得 RequestId）。定位问题时需要提供该次请求的 RequestId。|
