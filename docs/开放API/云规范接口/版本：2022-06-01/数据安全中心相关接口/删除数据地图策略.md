## 1. 接口描述

删除数据地图策略，私有化调用path为：/capi/DLPOpenApi/DeleteBackendPolicy，从8.6P1版本开始支持

## 2. 输入参数

| 参数名称 | 必选 | 类型 | 描述 |
|---------|---------|---------|---------|
| PolicyIds.N | 否 | Array of Integer | 策略列表ID<br/>示例值：[1] |

## 3. 输出参数

| 参数名称 | 类型 | 描述 |
|---------|---------|---------|
| Data | [DeleteBackendPolicyData](/开放API/云规范接口/版本：2022-06-01/数据结构.md#DeleteBackendPolicyData) | 业务响应数据<br/>注意：此字段可能返回 null，表示取不到有效值。|
| RequestId | String | 唯一请求 ID，由服务端生成，每次请求都会返回（若请求因其他原因未能抵达服务端，则该次请求不会获得 RequestId）。定位问题时需要提供该次请求的 RequestId。|

## 4. 示例

### 示例1 删除数据地图策略

#### 输入示例

```
POST / HTTP/1.1
Host: ioa.tencentcloudapi.com
Content-Type: application/json
X-TC-Action: DeleteBackendPolicy
<公共请求参数>

{
  "PolicyIds": [
    49
  ]
}
```

#### 输出示例

```json
{
    "Response": {
        "Data": {
            "PolicyIds": [
                49
            ]
        },
        "RequestId": "web-9929859776157-1765953461533"
    }
}
```
