## 1. 接口描述

展示自定义分组终端列表，私有化调用path为：/capi/Assets/Device/DescribeVirtualDevices，适用于私有化8.0以后的版本；8.0以前版本的调用path为：/capi/Assets/DescribeVirtualDevices。
