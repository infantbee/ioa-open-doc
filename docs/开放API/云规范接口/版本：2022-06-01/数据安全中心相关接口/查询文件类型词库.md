## 1. 接口描述

查询文件类型词库Id信息。私有化调用path为：capi/DLPOpenApi/DescribeDLPFileTypeName，从8.6P1版本开始支持

## 2. 输入参数

| 参数名称 | 必选 | 类型 | 描述 |
|---------|---------|---------|---------|
| DomainId | 否 | Integer | 管理域Id<br/>示例值：1 |

## 3. 输出参数

| 参数名称 | 类型 | 描述 |
|---------|---------|---------|
| Data | [DescribeDLPFileTypeNameData](/开放API/云规范接口/版本：2022-06-01/数据结构.md#DescribeDLPFileTypeNameData) | 文件类型数据|
| RequestId | String | 唯一请求 ID，由服务端生成，每次请求都会返回（若请求因其他原因未能抵达服务端，则该次请求不会获得 RequestId）。定位问题时需要提供该次请求的 RequestId。|

## 4. 示例

### 示例1 查询指定域所有文件类型

```
POST / HTTP/1.1
Host: ioa.tencentcloudapi.com
Content-Type: application/json
X-TC-Action: DescribeDLPFileTypeName
<公共请求参数>

{
  "DomainId": 1
}
```
